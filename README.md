These 2 Python mini scripts are to show you how you can leverage the new Radware consolidated cloud APIs to divert a DNS to the Cloud DDOS to protect it.
The undivert file is to deactivate the cloud diversion once the attack has been terminated.

Requires an API key that can be obtained from your account team.  You will also need the Infrastructure ID which can be copied from the portal.
Finally you will also need the asset ID which can also be obtained from the portal in the URL when the objected is selected.
Replace asset_endpoint variable with your asset ID from the portal.
